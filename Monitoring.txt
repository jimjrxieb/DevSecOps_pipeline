6. ** CONFIGURATION **
    > cd prometheus-3.0.0-beta.1.linux-amd64
    >> [vi prometheus.yml] 
>>> copy and paste []
--[
  - job_name: 'blackbox'
    metrics_path: /probe
    params:
      module: [http_2xx]  # Look for a HTTP 200 response.
    static_configs:
      - targets:
        - http://prometheus.io    # Target to probe with http.
        - https://<application-ip>:[k8-port]   # Target to probe with https
    relabel_configs:
      - source_labels: [__address__]
        target_label: __param_target
      - source_labels: [__param_target]
        target_label: instance
      - target_label: __address__
        replacement: <blackbox-IP>:9115  # The blackbox exporter's real hostname:port.
  ]--
  ** ADJUST ACCORDINGLY **
